# 台中市各區教育程度視覺化分析

### 簡介
1.台中市各區教育程度視覺化分析及各級學校分布。
2.使用Python處理資料，並使用Chart.js將資料視覺化。
3.使用Google Map API畫出台中市各區界線並顯示各級學校位置。
